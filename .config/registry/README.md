WindowsのキーをMac風に変える設定集
# regファイル
- caps_to_ctrl.reg：実行すると、CapsLockキーをCtrlキーに変換する
- default.reg：実行すると、デフォルトの状態に戻る

# AutoHotKey
キーバインドを仮想的に変換するアプリ：https://www.autohotkey.com/  
インストール後、KeyBind_MacLike.ahkを実行すると、Mac風のキー配列になる。  
[参考](https://tex2e.github.io/blog/keyboard/ahk-mac-like-keyboard)
